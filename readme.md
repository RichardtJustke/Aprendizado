# 📚 Repositório de Aprendizado

> Minha jornada de estudos em programação, documentada e organizada para construir uma base sólida de conhecimento.

---

## 🎯 Objetivo

Este repositório funciona como um **livro de receitas pessoal** onde centralizo todo meu aprendizado em programação. A ideia é consolidar conhecimentos de forma estruturada, permitindo consultas rápidas e acompanhamento da minha evolução como desenvolvedor.

---

## 📁 Estrutura do Repositório

```
Aprendizado/
├── 📝 Anotações/     → Documentação e conceitos teóricos
├── 💻 Linguagens/    → Exercícios práticos por linguagem
└── 🚀 Projetos/      → Projetos práticos e experimentações
```

---

## 📝 Anotações

Documentação técnica organizada por linguagem, cobrindo os **fundamentos universais da programação**:

| Conceito | Descrição |
|----------|-----------|
| Variáveis | Tipos de dados, declaração e escopo |
| Operadores | Aritméticos, lógicos e de comparação |
| Estruturas de Controle | Condicionais (if/else, switch) |
| Laços de Repetição | for, while, do-while |
| Funções | Declaração, parâmetros e retorno |
| Arrays/Listas | Manipulação de coleções |
| Objetos/Classes | Conceitos de POO |

> **Formato:** Um arquivo `.md` para cada linguagem, facilitando consultas rápidas quando surgir alguma dúvida.
