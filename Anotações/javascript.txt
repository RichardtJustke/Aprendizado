# O que é java script?
JavaScript é uma linguagem de programação leve, interpretada e orientada a objetos, amplamente utilizada para criar páginas web dinâmicas e interativas. Ela é executada no lado do cliente (navegador) e permite manipular elementos HTML, responder a eventos do usuário, validar formulários, criar animações, entre outras funcionalidades.

# tipos de dados
JavaScript possui vários tipos de dados, incluindo:
Números: Representam valores numéricos, tanto inteiros quanto de ponto flutuante.
Strings: Sequências de caracteres, usadas para representar texto.     
diferenciação entre strings e números
console.log("10" + 5); // Resultado: "105" (concatenação de string)
console.log(10 + 5);   // Resultado: 15 (soma numérica)
Booleanos: Representam valores verdadeiros (true) ou falsos (false).

# operadores aritimeticos
JavaScript suporta os seguintes operadores aritméticos:
Adição (+): Soma dois valores.  
Subtração (-): Subtrai um valor de outro.
Multiplicação (*): Multiplica dois valores.
Divisão (/): Divide um valor por outro.
Módulo (%): Retorna o resto da divisão entre dois valores.
Exemplo: 
let a = 10;
let b = 3;
console.log(a + b); // Resultado: 13
console.log(a - b); // Resultado: 7
console.log(a * b); // Resultado: 30
console.log(a / b); // Resultado: 3.3333...
console.log(a % b); // Resultado: 1
Operadores de atribuição
JavaScript suporta os seguintes operadores de atribuição:
Atribuição simples (=): Atribui um valor a uma variável.
Atribuição composta (+=, -=, *=, /=, %=): Combina uma operação
com a atribuição.
Exemplo:
let x = 10; // Atribuição simples
x += 5;     // Equivalente a x = x + 5
console.log(x); // Resultado: 15
x -= 3;     // Equivalente a x = x - 3
console.log(x); // Resultado: 12
x *= 2;     // Equivalente a x = x * 2
console.log(x); // Resultado: 24
x /= 4;     // Equivalente a x = x / 4
console.log(x); // Resultado: 6
x %= 4;     // Equivalente a x = x % 4
console.log(x); // Resultado: 2
Operadores de comparação
JavaScript suporta os seguintes operadores de comparação:
Igualdade (==): Verifica se dois valores são iguais (com conversão de tipo).
Estritamente igual (===): Verifica se dois valores são iguais (sem conversão de tipo
).Diferença (!=): Verifica se dois valores são diferentes (com conversão de tipo).
Estritamente diferente (!==): Verifica se dois valores são diferentes (sem conversão de tipo).
Maior que (>): Verifica se um valor é maior que outro.
Menor que (<): Verifica se um valor é menor que outro.
Maior ou igual (>=): Verifica se um valor é maior ou igual a outro.
Menor ou igual (<=): Verifica se um valor é menor ou igual a outro.
Exemplo:
let a = 10;
let b = "10";
console.log(a == b);  // Resultado: true (igualdade com conversão de tipo
console.log(a === b); // Resultado: false (estritamente igual sem conversão de tipo)
console.log(a != b);  // Resultado: false (diferença com conversão de tipo)
console.log(a !== b); // Resultado: true (estritamente diferente sem conversão de tipo)
console.log(a > 5);   // Resultado: true (maior que)
console.log(a < 15);  // Resultado: true (menor que)
console.log(a >= 10); // Resultado: true (maior ou igual)
console.log(a <= 5);  // Resultado: false (menor ou igual)
Operadores lógicos
JavaScript suporta os seguintes operadores lógicos:
E lógico (&&): Retorna true se ambos os operandos forem true.
Ou lógico (||): Retorna true se pelo menos um dos operandos for true.
Negação lógica (!): Inverte o valor lógico de um operando.
Exemplo:
let x = 10;
let y = 5;
console.log(x > 5 && y < 10); // Resultado: true (E lógico)
console.log(x > 15 || y < 10); // Resultado: true (Ou lógico)
console.log(!(x > 5)); // Resultado: false (Negação lógica)
